Privacy Policy

Effective Date: July 9, 2025

This Privacy Policy explains how the Drift Chrome extension ("Drift") handles your data. Drift does not collect, store, or transmit any personal information, except as needed to deliver its core functionality.

1. What Drift Does

Drift helps you summarize, translate, and ask questions about content on webpages and local files using Google’s Gemini API. To do this, Drift:

  • Uses your text input or microphone input (only if you grant permission) to create prompts.
  • Temporarily processes this input and sends it to Gemini AI for a response.
  • Stores your Gemini API key locally in Chrome’s extension storage to authenticate requests. It is never sent to any other server.

2. What Drift Does NOT Do

  • Drift does not store or log your text or voice input after it’s used in a prompt.
  • Drift does not transmit any data to third-party servers other than Gemini API.
  • Drift does not use tracking, analytics, or advertising cookies.
  • Drift does not require you to create an account or log in.

3. Microphone Access

If you choose to use voice input, Chrome will request your permission to access your microphone. Drift only uses this to convert speech to text. No audio is recorded or stored.

4. Data Security

Your API key and settings are stored securely in Chrome’s local storage. Drift runs entirely in your browser’s sandbox environment and follows Chrome Web Store security best practices.

5. Changes

If there are any significant changes to this policy, we will update this page and change the “Effective Date.”
